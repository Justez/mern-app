# MERN app

Required ENVs: 

`DB_LOGIN`, `DB_PASSWORD` for mongo db cloud [https://cloud.mongodb.com]

`RECIPE_APP_ID`, `RECIPE_APP_SECRET` for [https://api.edamam.com]


Start client: 
`cd client && npm start`

Start server: 
`DB_LOGIN=<> DB_PASSWORD=<> RECIPE_APP_ID=<> RECIPE_APP_SECRET=<> npm start`
